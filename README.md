# 🎓 Student Management System

Sistema de gerenciamento de estudantes desenvolvido em **Java**, com foco na prática de **Programação Orientada a Objetos (POO)** e operações básicas de manipulação de dados (CRUD).

---

## 🚀 Sobre o Projeto

Este projeto permite gerenciar estudantes de forma simples e eficiente, realizando operações como cadastro, listagem, atualização e remoção de alunos.

Foi desenvolvido com o objetivo de reforçar conceitos fundamentais de Java e organização de código.

---

## 🛠️ Tecnologias Utilizadas

- ☕ Java  
- 📦 Programação Orientada a Objetos (POO)  
- 🗂️ Estruturas de dados  

---

## ⚙️ Funcionalidades

- ➕ Cadastrar estudantes  
- 📄 Listar estudantes  
- ✏️ Atualizar dados  
- ❌ Remover estudantes  
- 🔍 Buscar estudantes
