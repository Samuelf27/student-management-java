# 🎓 Student Management System

Sistema de gerenciamento de estudantes desenvolvido em **Java**, com foco em organização, boas práticas de programação e aplicação de conceitos de **Programação Orientada a Objetos (POO)**.

---

## 🚀 Sobre o Projeto

Este projeto tem como objetivo gerenciar informações de estudantes de forma simples e eficiente, permitindo operações como cadastro, listagem, atualização e remoção de dados.

É um projeto voltado para prática de lógica de programação e estruturação de aplicações em Java.

---

## 🛠️ Tecnologias Utilizadas

- ☕ Java
- 📦 Programação Orientada a Objetos (POO)
- 🗂️ Estruturas de dados (Listas, Arrays, etc.)

---

## ⚙️ Funcionalidades

- ➕ Cadastrar estudantes  
- 📄 Listar estudantes  
- ✏️ Atualizar dados  
- ❌ Remover estudantes  
- 🔍 Buscar estudante
